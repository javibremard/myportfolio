BLUEj:
PROCEDURAL MAGAZINE

web, editorial

See Proyect ↰

BLUEj is an editorial project that aims to rescue the imaginary and visual style of "pulp" magazines and adapt it to the new procedural publishing formats.

Due to the social trend towards rapid consumption of new themes and aesthetics, and in order to offer different approaches in each issue, this project is proposed with the aim of making it easier to develop the following issues of the magazine.

In this way, BLUEj becomes a procedural graph, offering a visual system flexible enough to adapt to the following numbers through a system of tools with which to compose the graphic system.

PICKER:
CALENDAR APP

prototype, UX / UI

See Proyect ↰

Project in collaboration with Manuel Temprado and Celia López.

Picker is an interactive application designed to plan tasks and events, working as a calendar and academic agenda.

Its main concept is the functional use of colour, customization and its accessibility, giving character to a virtual assistant that guides the user through the application, organizes the calendar and makes the experience more enjoyable.

The application claims to be fully accessible for any dyschromatopsia (inability or difficulty to discern colors). It allows configuring up to nine different display modes that make it easier for users with color blindness and achromatopsia to read it, configuring the white contrast and replacing colors with visual patterns.

БОСИКОМ:
SWAN LAKE FESTIVAL

branding

БОСИКОМ [barefoot] is a personal project for the premiere of a new version of the famous ballet Swan Lake (Pyotr Ilyich Tchaikovsky).

The main element of the graphic is a 3D mechanical swan, fusing the more literal aspect of the ballet and Russian cyberpunk culture.

This mechanical swan represents the mechanical precision and elegance of the ballet discipline.

The intention of this project is to attract the attention of the public and experiment by accentuating illegibility; with the aim of causing visual stress that is both attractive and interesting.

OUT OF WATER|
MERMAIDS

ilustrations

In order to recreate the look of comic illustrations with 3D materials, these illustrations are intended to evoke a narrative in the form of social complaint.

With a mermaid as the main character, this series aims to raise awareness about violence against women and the oceans.

Both represented in the figure of mermaids as victims of current mass consumerism, pollution and beauty standards.

ANASTASIA

3D design

3D Model ↰

This is my 3D version of Anya from the 1997 animated film "Anastasia."

This project has aimed to translate an existing 2D design into a three-dimensional model, trying to be as faithful as possible to Don Bluth's original design.

The body have been modeled and sculpted in Cinema4D and Blender.

The outfit has been made in Marvelous Designer.
